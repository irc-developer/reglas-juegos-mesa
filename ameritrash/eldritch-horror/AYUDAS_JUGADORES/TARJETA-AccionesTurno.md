## FRENTE (Cara A)

```
═══════════════════════════════════════════════════════
              ACCIONES DE TURNO - FASE 1
           (Máx 2 acciones / 1 tipo por ronda)
═══════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────┐
│ VIAJAR                                              │
├─────────────────────────────────────────────────────┤
│ Req: Ruta conectada                                 │
│ Efecto: Mueve 1 casilla                             │
│ Extra: +1 billete (Tren/Barco) = +1 casilla        │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│ DESCANSAR                                           │
├─────────────────────────────────────────────────────┤
│ Req: ✗ Monstruo en casilla                          │
│ Efecto: +1 SAL / +1 COR (de reserva)                │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│ INTERCAMBIAR                                        │
├─────────────────────────────────────────────────────┤
│ Req: Inv. en tu casilla                             │
│ Efecto: Intercambia posesiones                      │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│ PREPARARSE VIAJAR                                   │
├─────────────────────────────────────────────────────┤
│ Req: Casilla CIUDAD (no nº)                         │
│ Efecto: +1 billete (eliges Tren/Barco)              │
│ Límite: Max 2 billetes en mano (descarta exceso)    │
└─────────────────────────────────────────────────────┘
```

├─────────────────────────────────────────────────────┤

│ Límite: 1x por ronda por FUENTE diferente           │
│ Efecto: Lo que dice el componente                   │
└─────────────────────────────────────────────────────┘

╔═════════════════════════════════════════════════════╗
║              CASOS ESPECIALES                       ║
╠═════════════════════════════════════════════════════╣
║ • Monstr. en casilla → ✗ Descansar / ✗ Apoyos      ║
║ • 3 billetes → descartar 1 si usas Prep. Viajar   ║
║ • Acción NO puede repetirse en misma ronda         ║
║   (sí cambiar tipo: Viajar + Descansar OK)         ║
╚═════════════════════════════════════════════════════╝

## Notas Técnicas (para edición)

**Tamaño recomendado de impresión**:
- Formato: A6 (10.5 x 14.8 cm aprox)
- O: Tarjeta de tarot estándar 12 x 9 cm
- Calidad: 300 dpi mínimo
- Color: A color o B/N (ambos legibles)

**Fuente usada en este ejemplo**:
- Texto principal: Courier New 6.5pt
- Títulos: Courier New 7pt **bold**
- Símbolos de caja: Compatible ASCII

**Margen seguro**: 0.5 cm mínimo en bodega

**Para edición futura**:
- Mantener estructura de dos caras (frente/dorso)
- Validar cada cambio contra Eldritch-Horror.md
- Si crece contenido: crear segunda tarjeta (Combate, Encuentros, etc.)

2. **Plastificar o forrar** para durabilidad (opcional pero recomendado)
3. **Colocar junto a tablero** dentro del área de juego
4. **Referencia rápida**: Si hay duda → consultar tarjeta antes que rulebook
5. **Para excepciones complejas**: Va a Eldritch-Horror.md completo
