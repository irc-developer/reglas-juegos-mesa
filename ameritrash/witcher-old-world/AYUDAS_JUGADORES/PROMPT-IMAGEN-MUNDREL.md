# Prompt de imagen para Mundrel — The Witcher: El Viejo Mundo

## Objetivo

Generar una ayuda visual vertical, muy legible y útil durante la partida. Debe evocar una aventura medieval de caza de monstruos sin copiar arte, logotipos, marcos, iconos ni ilustraciones oficiales.

**Fuente mecánica del contenido**: manual local en español, pp. 9-29 y 35-36.

---

## Versión 1 — Ayuda visual de una página

### Prompt listo para pegar

Crea una ayuda visual vertical de una página para un juego de mesa de aventura, exploración y caza de monstruos titulado «El Viejo Mundo — Ayuda de turno y combate».

La pieza debe ser una referencia de mesa altamente legible, no un póster ni una portada. Formato A4 o Carta vertical, composición editorial modular, alto contraste, texto breve, flechas claras, cajas compactas y jerarquía visual fuerte. Prioriza siempre la utilidad de juego sobre la decoración.

Inspiración visual general: fantasía oscura medieval, pergamino envejecido claro, mapa antiguo, cuero oscuro, metal usado, frascos de alquimia, huellas de monstruo y marcas de espada. La estética debe sentirse compatible con un mundo de cazadores de monstruos, pero ser completamente original.

No copies ni reproduzcas arte oficial, personajes reconocibles, logotipos, tipografías de marca, iconos, símbolos, cartas, marcos o ilustraciones propietarias. No uses el logotipo oficial de The Witcher. Usa iconografía genérica y original.

Organiza la página en estos bloques, de arriba abajo:

1. CABECERA
«EL VIEJO MUNDO»
«AYUDA DE TURNO Y COMBATE»

2. OBJETIVO, en una franja destacada
«4 TROFEOS → VICTORIA»
«El último debe venir de un combate»

3. TURNO DEL JUGADOR, como flujo central
«I. MOVER Y ACTUAR»
«Mover → Acción local / Póquer / Misión → Repetir o seguir»
«II. ELEGIR 1»
«Combatir | Meditar | Explorar»
«III. CARTAS»
«Descartar opcional → Robar hasta 3 → Obtener 1 obligatoria»

4. MOVIMIENTO, en una caja compacta
«1 carta del Terreno»
«O 2 cartas cualesquiera»
«O 1 carta + 1 Oro»
«O 1 Comodín»
«→ Localización conectada»
Aviso: «Muévete antes de usar una Acción local»

5. DECISIÓN DE FASE II
«Combatir: monstruo o brujo aquí»
«Meditar: atributo 5 + trofeo disponible»
«Explorar: Ciudad o Tierras Salvajes»

6. CARTAS
«Conserva máximo 3 → Roba hasta 3»
«Obtén 1 carta obligatoriamente»
«Coste = descartes de mano»
«Derecha -1 | Izquierda +1»

7. COMBATE
«1. Pociones / Especialidad / efectos»
«2. Combo por colores»
«3. Daño → Escudo → efectos»
«4. Roba: Ataque + modificadores»

8. DAÑO
«A BRUJO: Escudo → Mazo → Mano → Noqueado»
«A MONSTRUO: 1 Daño = -1 carta de Vida»

9. MONSTRUO
«Rastro: empiezas tú»
«Sin Rastro: empieza el monstruo»
«Embestida/Mordisco → revelar → aplicar»

10. LÍMITES CRÍTICOS, en una franja inferior
«MANO 7»
«POCIONES POSEÍDAS 4»
«USOS MÁX. DE POCIÓN = ALQUIMIA»
«ESCUDO ≤ DEFENSA»
«ACCIÓN LOCAL 1×/TURNO»
«MOVER ANTES DE ACTUAR»
«MEDITAR NO CIERRA PARTIDA»

Usa frases cortas exactamente como aparecen. No inventes reglas, valores, nombres de acciones ni texto narrativo. Si una frase no cabe con buena legibilidad, reduce decoración antes que tipografía.

Paleta: marfil y pergamino para fondo, negro carbón para texto, rojo oscuro para Daño, verde apagado para Alquimia, azul grisáceo para Movimiento y dorado envejecido para Trofeos. Tipografía limpia y muy legible; títulos con serif medieval sobria y reglas con sans serif clara.

Resultado final: infografía vertical premium, ordenada, imprimible, sin mockup, sin manos y sin fondo escénico dominante.

## Versión 2 — Dos tarjetas: turno + combate

### Prompt listo para pegar

Crea dos tarjetas verticales coordinadas de ayuda para un juego de mesa de fantasía oscura medieval y caza de monstruos. Deben compartir paleta, tipografía, bordes y sistema de iconos originales. Alta legibilidad, composición compacta, frases mínimas y uso inmediato en mesa.

No copies arte, logotipos, tipografías, cartas, marcos, iconos, personajes ni ilustraciones oficiales. No uses el logotipo oficial de The Witcher. La temática debe ser compatible, pero el diseño y todos los elementos gráficos deben ser originales.

TARJETA A — TURNO DEL JUGADOR

Cabecera:
«EL VIEJO MUNDO»
«TURNO DEL JUGADOR»

Objetivo:
«4 TROFEOS → VICTORIA»
«El último debe venir de un combate»

Flujo:
«I. MOVER Y ACTUAR»
«Mover → Acción local / Póquer / Misión → Repetir»
«II. ELEGIR 1»
«Combatir | Meditar | Explorar»
«III. CARTAS»
«Descartar opcional → Robar hasta 3 → Obtener 1 obligatoria»

Movimiento:
«Terreno: 1 carta»
«Cualquiera: 2 cartas»
«Cualquiera: 1 carta + 1 Oro»
«Comodín: 1 carta»

Recordatorios:
«Muévete antes de usar Acción local»
«Acción local: 1×/turno»
«Mano máxima: 7»
«Derecha -1 | Izquierda +1»
«Póquer impide atacar a ese brujo»

TARJETA B — COMBATE

Cabecera:
«EL VIEJO MUNDO»
«COMBATE»

Preparación:
«Conserva mano»
«Mazo + descartes = Mazo de Vida»
«Monstruo: Reserva = valor de Vida»
«Rastro: empiezas tú»

Turno de brujo:
«1. Pociones / Especialidad / efectos»
«2. Combo por colores»
«3. Daño → Escudo → efectos»
«4. Roba: Ataque + modificadores»

Daño:
«Brujo: Escudo → Mazo → Mano → Noqueado»
«Monstruo: 1 Daño = -1 carta de Vida»

Turno de monstruo:
«Embestida/Mordisco → revelar → aplicar»

Resultados mínimos:
«Derrotar: trofeo + 2 Oro + Fatiga»
«Ahuyentar: 2 Oro + carta coste 0»
«Derrota: Rastro si falta + carta coste 0 + roba hasta 2»

Recordatorios:
«Escudo ≤ Defensa»
«4 Pociones poseídas»
«Usos máximos por combate = Alquimia»
«Fatiga tras trofeo»
«Reconstruye mazos y restaura Escudo»

Estilo de ambas tarjetas: pergamino claro, bordes de cuero y metal muy discretos, pequeños motivos originales de mapa, alquimia y armas. Nada debe competir con el texto. Usa iconos genéricos y originales para movimiento, Oro, carta, Escudo, Daño, Poción y Trofeo.

No inventes ninguna regla. Mantén exactamente las frases indicadas y reduce decoración si hace falta espacio.

## Referencias visuales opcionales

Si se adjuntan imágenes, úsalas únicamente para comprender:

- proporción y densidad de una ayuda de mesa;
- legibilidad de tablas y flechas;
- textura general de pergamino, cuero o mapa medieval.

No copies de las referencias ningún logotipo, personaje, ilustración, icono, carta, marco ni composición protegida.

---

## Negative prompt

Evitar:

- texto diminuto o ilegible;
- párrafos largos;
- exceso de decoración;
- ruido visual;
- jerarquía débil;
- fondo demasiado oscuro;
- ilustración dominante;
- aspecto de póster o portada;
- logotipo oficial;
- tipografía oficial o imitada;
- arte de cartas copiado;
- marcos o iconos copiados;
- personajes oficiales reconocibles;
- ilustraciones propietarias;
- reglas falsas o inventadas;
- valores distintos de los indicados;
- exceso de texto de ambientación;
- interfaz futurista;
- neón o estética cyberpunk;
- estilo cartoon o anime;
- mockup de caja;
- manos humanas;
- marcas de agua;
- texto deformado.

---

## Ajustes recomendados

- Orientación: vertical.
- Versión de página: A4 o Carta, 300 dpi.
- Versión de tarjetas: proporción aproximada 7:12, 300 dpi.
- Fidelidad al texto: máxima.
- Decoración: baja o media.
- Contraste: alto.
- Prioridad: legibilidad y utilidad durante la partida.
